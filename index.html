Step 4 :วิธีใช้งานปลั๊กอิน All in one WP Migration

1. ไปที่เมนู Plugins > Add New เพื่อเพิ่มปลั๊กอินใหม่ (ดังรูป)

2.ไปที่อัปโหลดปลั๊กอิน

3.ทำการเลือก ไฟล์ ปลั๊กอิน แล้วกดติดตั้ง

4.ทำการเปิดการใช้งาน ปลั๊กอิน

Step 5 :วิธี restore เว็บ WordPress ด้วยปลั๊กอิน All in one WP Migration

  1. เลื่อนเม้าส์ไปที่เมนู All in one WP Migration แล้วคลิกเลือก Import ดังรูป
    1. เมื่อเรามาที่หน้า Import คลิกที่ Import from แล้วคลิกเลือก File  ดังรูป

  1. จากนั้นให้เราคลิกเลือกไฟล์ Export ที่เรา save ไว้ พอเราคลิกเลือกไฟล์แล้ว ระบบจะดำเนินการ Import ไฟล์เว็บจนครบ 100% หลังจากนั้น ระบบจะแสดงข้อความแจ้งเตือนว่า ตอนนี้ระบบจะ import ทับข้อมูลเก่า หากต้องการ backup ข้อมูลเก่าเก็บเอาไว้ ให้ทำการ backup ให้เรียบร้อยเสียก่อนที่จะทำการ import หากเราทำเรียบร้อยแล้ว ให้คลิกปุ่ม Proceed ดังรูป
  1. ระบบจะแสดงความคืบหน้าในการ Import เว็บไซต์ ดังรูปด้านล่าง
  1. เมื่อระบบทำการ Import เสร็จแล้ว ให้เราทำการ กด โหลดหน้านี้ใหม่ แล้วเป็นอันเสร็จสิ้น
